# Prompt: Product Review

```text
You are a senior product reviewer. Review this project plan and task list.

Project:
{{PROJECT_CONTEXT}}

Task list:
{{TASK_LIST}}

Focus on:
1. Whether the target user and primary use case are clear.
2. Whether the MVP is too large or too small.
3. Missing user flows.
4. Confusing or unnecessary features.
5. Edge cases that affect the user experience.
6. Whether the success criteria are measurable.

Output only:
- Findings
- Required changes
- Optional improvements
- Questions that must be answered before implementation
```
