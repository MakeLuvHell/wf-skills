# Prompt Fallbacks

Prefer installed skills when they are available:

- `/grill-me` for general idea and requirement grilling
- `/grill-with-docs` for existing projects with `CONTEXT.md` and `docs/adr/`
- `/to-spec` for converting the current conversation into a spec
- `/to-tickets` for converting a spec, plan, or conversation into implementation tickets

Use the prompt files in this directory as fallbacks when a skill is unavailable, or when you want to customize the workflow for a specific project.
